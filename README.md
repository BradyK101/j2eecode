 # j2eecode.zip包含的内容  

这个版块里边包含：java，jdbc，servlet，jsp，spring，springmvc，mybatis，redis等技术的全部学习代码。 

# j2eecode.zip下载方法

点击“j2eecode.zip” ，进入之后，点击Download按钮即可

# 压缩包解压后的文件夹介绍：  

压缩包解压后，打开文件夹，然后可以看到一个一个子文件夹，每个子文件夹是一个代码工程。你需要将每个工程导入到eclipse中即可使用
